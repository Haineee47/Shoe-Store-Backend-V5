# Shoe Store Backend

Backend service for the Shoe Store e-commerce platform.

## Project Status

The repository is currently in the foundation setup phase.

## Planned Technology Stack

- Java 21
- Spring Boot
- Maven
- PostgreSQL
- Flyway
- Spring Data JPA
- Spring Security
- Docker
- GitHub Actions

## Architecture

The project follows a modular monolith architecture with practical Domain-Driven Design principles.

## Modules

The initial business modules are expected to include:

- Identity
- Catalog
- Inventory
- Cart
- Order
- Payment

## Development Status

No business module has been implemented yet.
